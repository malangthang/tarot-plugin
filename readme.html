Tarot Pro Plugin

Tính năng

  • Custom Post Type cho quản lý lá bài Tarot
  • Meta box cho ý nghĩa theo ngữ cảnh (upright/reversed)
  • SEO settings cho từng lá bài
  • Import 78 lá bài từ file JSON
  • Rút bài ngẫu nhiên với AI phân tích
  • Shortcode [tarot_draw] để hiển thị form rút bài
  • Admin interface sử dụng native WordPress

Cài đặt

  1. Upload plugin vào thư mục wp-content/plugins/
  2. Kích hoạt plugin trong WordPress admin
  3. Đi đến Tarot > Settings để cấu hình OpenAI API key
  4. Đi đến Tarot > Import để import 78 lá bài

Cách sử dụng admin

Quản lý lá bài

Đi đến Tarot Cards trong menu admin để quản lý từng lá bài:

  • Title: Tên lá bài (vd: The Fool)
  • Excerpt: Mô tả ngắn (lưu từ description trong JSON)
  • Content Editor: Viết nội dung chi tiết về lá bài
  • Featured Image: Ảnh lá bài
  • Card Meaning meta box:
    • Ý nghĩa Xuôi (Upright)
    • Ý nghĩa Ngược (Reversed)
  • SEO Settings meta box: SEO Title, Description, Keywords

Import lá bài

Đi đến Tarot > Import để import toàn bộ 78 lá bài từ file JSON.

Cấu trúc dữ liệu

Tarot Card Post (WordPress native)

  • post_title = Tên lá bài
  • post_excerpt = Description (mô tả ngắn)
  • post_content = Nội dung chi tiết
  • post_thumbnail = Ảnh lá bài
  • post_meta _tarot_upright = Ý nghĩa xuôi
  • post_meta _tarot_reversed = Ý nghĩa ngược
  • post_meta _tarot_arcana = Arcana type
  • post_meta _tarot_seo_* = SEO settings

Bảng tarot_cards (backup query)

  • Được tạo từ import, dùng để query nhanh
  • Chứa name, slug, arcana, image, description, upright_meaning, reversed_meaning

Bảng tarot_readings (history)

  • card_id, question, context, position, ai_text, created_at

API Settings

Cần cấu hình OpenAI API key để sử dụng tính năng AI phân tích. Lấy key từ OpenAI Platform.

Database Tables

  • wp_tarot_cards - Thông tin cơ bản lá bài
  • wp_tarot_card_meanings - Ý nghĩa theo ngữ cảnh
  • wp_tarot_readings - Lịch sử rút bài
